FeedMe Plugin Documentation

Description

The FeedMe plugin allows players on your Minecraft server to feed each other with various food items. Use cases such as players who are AFK taking damage from hunger allows other players to feed them.

Features

  • Players can feed each other with various food items.
  • Prevents feeding harmful foods such as raw chicken and pufferfish.
  • Special effects for items like golden apples.
  • Cooldown system to prevent spamming of feeding actions.

Commands

No commands in this plugin.

Permissions

No permissions required.

Configuration

No configuration required. The plugin is ready to use after installation.

Very simple for users.

Usage

To feed another player, simply right-click them while holding a valid food item.

Golden apples will provide regeneration and absorption effects.

Compatibility

This plugin is compatible with Spigot, Paper, Bukkit, and more.
